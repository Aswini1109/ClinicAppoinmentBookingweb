# Clinic Appointment Booking Web

A simple web app for clinics to manage patient appointments online.

## Features
- Patient can book appointment with date & time
- Admin can view all bookings in admin.html  
- Mobile responsive design

## Tech Stack
HTML, CSS, JavaScript
